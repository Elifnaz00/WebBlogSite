# Asp .Net 5.0 MVC5 BlogSite
## Kullanılan Teknolojiler ve Başlıklar
-Entity Framework, 
<br/>
-N Katmanlı Mimari, 
<br/>
-Repository Design Pattern, 
<br/>
-Layout Kontrolü, 
<br/>
-Code First,
<br/>
-Linq Sorguları, 
<br/>
-Trigger Kullanımı,
<br/>
-İlişkili Tablolar ve Diyagramlar, 
<br/>
-Partial View, 
<br/>
-CSS Kontrolleri,
<br/>
-Bootstrap,
<br/>
-Prosedür Oluşturma, 
<br/>
-Dashboard yapısı
<br/>
-Hazır Template Kurulumu ve Düzenlenmesi
<br/>
